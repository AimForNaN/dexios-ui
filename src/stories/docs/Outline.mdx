# dexios-ui-outline-*

## dexios-ui-outline-inner

Used to apply an inset ring that is one pixel wide by default and ring color.
Uses arbitrary values for specifying color, shade and ring width.
The ring color will take on the color of the arbitrary value.

Expected syntax for arbitrary value is in the `[color-shade_width]` form.

### Examples

```css
ui-outline-inner-[zinc-300]
ui-outline-inner-[zinc-300_4]
ui-outline-inner-[black_2]
```

## dexios-ui-outline-outer

Used to apply an outline that is one pixel wide by default and outline color.
Uses arbitrary values for specifying color, shade and outline width.
The outline color will take on the color of the arbitrary value.

Expected syntax for arbitrary value is in the `[color-shade_width]` form.

### Examples

```css
ui-outline-outer-[zinc-300]
ui-outline-outer-[zinc-300_4]
ui-outline-outer-[black_2]
```
