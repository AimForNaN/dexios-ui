# dexios-ui-outline

Used to apply an ring that is one pixel wide by default and a ring color with matching text color.
Uses arbitrary values for specifying color and shade of outline.
Ring color will take on the color of the arbitrary value.
Regardless of the shade specified, the matching text color will always be of the shade 950.
If the color specified has no shade, a text color without shade will be chosen.

Expected syntax for arbitrary value is in the `[color-shade_width]` form.

### Examples

```
dexios-ui-outline-[zinc-300]
```
