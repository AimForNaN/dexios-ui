# ui-solid

Used to apply a background color and matching text color.
Uses arbitrary values for specifying color and shade.
The background color will take on the color of the arbitrary value.

Expected syntax for arbitrary value is in the `[color-shade]` form.

### Examples

```css
ui-solid-[zinc-300]
ui-solid-[black]
```
