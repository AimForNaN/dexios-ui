# dexios-ui-outline

Used to apply a background with a shade of 50 and a matching text color with a shade of 700.
Uses arbitrary values for specifying a base color.
We recommend a shade of 800 for text requiring emphasis for soft variant.

Expected syntax for arbitrary value is in the `[color]` form.

### Example

```css
ui-soft-[zinc]
```
