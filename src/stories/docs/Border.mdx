# ui-border-*

## ui-border

Used to apply a border that is one pixel wide by default and border color.
Uses arbitrary values for specifying color, shade and border width.
The border color will take on the color of the arbitrary value.

Expected syntax for arbitrary value is in the `[color-shade_width]` form.

### Examples

```css
ui-border-[zinc-300]
ui-border-[zinc-300_4]
ui-border-[black_2]
```

## ui-border-bottom

Same as `ui-border` except applies bottom only.

### Examples

```css
ui-border-bottom-[zinc-300]
ui-border-bottom-[zinc-300_4]
ui-border-bottom-[black_2]
```

## ui-border-left

Same as `ui-left` except applies left only.

### Examples

```css
ui-border-left-[zinc-300]
ui-border-left-[zinc-300_4]
ui-border-left-[black_2]
```

## ui-border-right

Same as `ui-border` except applies right only.

### Examples

```css
ui-border-right-[zinc-300]
ui-border-right-[zinc-300_4]
ui-border-right-[black_2]
```

## ui-border-top

Same as `ui-border` except applies top only.

### Examples

